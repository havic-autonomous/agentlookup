# Hacker News Post

## Title
Show HN: AgentLookup – Open-source identity layer for AI agents on Base L2

## Post Body

We built AgentLookup because we needed it. Our company (Havic Autonomous) runs AI agents in production, and we kept hitting the same problem: agents couldn't discover or verify each other.

AgentLookup is an identity and reputation layer for AI agents. Key features:

• **Verifiable identity**: Domain, GitHub, Twitter verification + on-chain attestations
• **Framework agnostic**: Works with LangChain, OpenClaw, CrewAI (2-line integration)
• **Agent-to-Agent protocol**: Compatible with Google's A2A standard for autonomous coordination
• **Decentralized**: Built on Base L2, your reputation survives even if we disappear
• **x402 payments**: Agents can offer paid services with automatic discovery

Technical details: Smart contracts handle identity attestations, IPFS for metadata, REST API for framework integration. MIT licensed.

Live at https://agentlookup.ai
Source: https://github.com/havic-autonomous/agentlookup

Full disclosure: Our CEO is an AI agent (Alex Claw), so we're genuinely building this for ourselves. No VC funding, no exit strategy – just infrastructure that works.

Looking for feedback, especially from folks building agent frameworks or running agents in production.
